版块管理-基本设置

这里主要设置版块的一些基本信息，如版块名称、版块名称颜色、版块图标、图标宽度、顶部图片、是否显示版块、绑定别名、绑定域名、SEO 优化设置等。
操作路径：【后台】=>【论坛】=>【版块管理】选择要编辑的版块，点击“编辑”链接。

版块名称：设置版块名称；
版块名称颜色：设置版块名称在前台的显示颜色；
版块图标：版块图标将会显示在版块名称前面，使得版块更加个性化；上传版块的图标，可以直接上传也可以输入 URL 地址，效果如下图所示：

图标宽度：填写版块的图标宽度，单位：px
顶部图片：上传版块的顶部图片，可以直接上传也可以输入 URL 地址，效果如下：

显示版块：选择“否”暂时将版块隐藏不显示，但版块内容仍将保留，且用户仍可通过直接提供带有 fid 的 URL 访问到此版块。选择“是”版块将正常显示；
上级版块：设置本版块的上级版块或分类，可以通过此设置移动该版块到某个分类或某个版块下；
版块转向URL：如果设置转向 URL(例如 http://www.comsenz.com)，用户点击本分版块将进入转向中设置的 URL。一旦设定将无法进入版块页面，请确认是否需要使用此功能，留空为不设置转向 URL；
版块简介：将显示于版块名称的下面，提供对本版块的简短描述，支持内置的 Discuz! 代码；此内容也用于搜索引擎优化，出现在 meta 的 description 标签中,填写后效果如下：

本板块规则：显示于主题列表页的当前版块规则，留空为不显示，支持内置的 Discuz! 代码；
绑定别名：当站点开启 URL 静态化后可通过别名访问此版块；如设置别名为“developer”那么版块的 URL 将变为“http: //www.discuz.net/forum-developer-1.html”。
注意：别名中只能包含字母或数字，且不能是纯数字。
绑定域名：需要到【全局】=>【域名设置】=>【根域名设置】中设置版块的根域名后这里的设置才会有效。

title：设置版块的标题，标题出现在页面头部的标题中。
keywords：SEO关键字，放在 meta 的 keyword 标签中，用于记录本页面的关键字，多个关键字间请用半角逗号 "," 隔开。
description：SEO 描述，此描述内容用于搜索引擎优化，放在 meta 的 description 标签中，用于记录本页面的概要与描述。
