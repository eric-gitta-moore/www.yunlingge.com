管理组

管理组包括管理员、超级版主、版主三个内置管理组以及自定义管理组，站点内所有和管理组权限有关的设置，都要在这里进行开启或者关闭，下面以超级版主为例，讲解一下管理组的各项设置。
操作路径：【后台】=>【用户】=>【管理组】

一、基本设置

点击超级版主的“基本设置”链接，如下图所示：

1、基本设置
1)用户组头衔：设置用户组头衔名称。
2)组图标：设置该用户组的图标，可以选择是输入 URL 或者直接上传图片。
组图标设置好以后会在以下几个地方显示
a、显示在用户头像下方

b、显示在家园首页个人头像处

c、显示在好友列表中

等等。
3)访问站点权限：
禁止访问：禁止用户浏览本站任何页面。
正常访问：当站点开放的时候，可以正常浏览页面。
超级访问：即便站点关闭也可以访问，一般用于管理员或站内测试组。
4)阅读权限：设置用户浏览帖子或附件的权限级别，范围 0～255，0 为禁止用户浏览任何帖子或附件。当用户的阅读权限小于帖子或附件的阅读权限许可(默认时为 1)时，用户将不能阅读该帖子或下载该附件。
5)最多好友数：该用户组的用户能够拥有的最多好友数量，0为无限。
除了用户组的好友数限制外，如果想单独给某个管理组的成员单独增加好友数，可以到【用户】=>【用户管理】=>【详情】中添加额外好友数。
6)二级域名最短长度：站点开启二级域名功能后有效，0为禁止使用二级域名。
7)允许隐身：是否允许用户在站点上隐身。
8)允许加入/离开公众用户组：是否允许该用户组用户在站点加入或者离开公众用户组。
9)允许积分转账：设置是否允许用户在个人中心将自己的交易积分转让给其他用户。注意：本功能需在 全局设置中启用交易积分后才可使用。
10)允许发送短消息：设置是否允许用户发送短消息。选择“是”后可以设置24小时内发布短消息的最大数及是否可以给任何人发短消息，选择不允许给任何人发短消息，那么选择“否”当对方设置为只接受好友短消息，将无法对其发送短消息。
11)允许使用 HTML 代码：是否允许用户发布信息的时候使用 HTML 代码。注意：开放 HTML 功能将产生安全隐患，请慎用。建议只在十分必要的情况下使用，并限制只开放给最核心的管理人员。
12）允许发站外URL：该项设置可以有效减少广告帖子。
13)允许查看站点统计：是否允许用户查看站点统计。选择“是”用户在访问站点的时候页面底部会出现“站点统计”链接，点击即可查看。

14)允许全文搜索：是否允许用户进行全文搜索，此设置只对搜索论坛有效。注意：当数据量大时，全文搜索将非常耗费服务器资源，请慎用。
15)允许使用搜索：选择为该用户组用户开启哪些模块的搜索。包括搜索文章、论坛、日志、相册和群组。
16)操作理由短消息通知作者：设置用户在对他人评分或管理操作时是否强制输入理由和通知作者。
17)允许自定义头衔：设置是否允许用户设置自己的头衔名字并在帖子页面中显示。选择“是”用户在【个人资料】中设置自己的头衔后效果如下：

18)不受时间段限制：设置用户是否可以不受全局设置中设置的特殊时间段禁令限制。
19)会员每小时发主题数限制：设置允许会员每小时最多的发表主题数量，可以配合灌水预防功能进一步限制会员的发帖，可设置为 1～255 范围内的数值，0 为不限制。此功能会轻微加重服务器负担, 且对游客无效。
20）会员每小时发回帖数限制：设置允许会员每小时最多的发回帖数量，可以配合灌水预防功能进一步限制会员的发帖，可设置为 1～255 范围内的数值，0 为不限制。此功能会轻微加重服务器负担，且对游客无效
21)启用防灌水验证码或验证问答机制：开启防灌水验证机制可以防止灌水机等，但会增加用户操作易用度。修改密码不受此限制，开启防灌水验证码或验证问答后，该设置有效。
22)发表不受限制：设置发表是否不受灌水预防和最大字数限制等。
23)忽略需要审核的关键字：选择“是”则此用户组的用户在发表时不会触发需要审核的关键字。
24）允许用户创建淘专辑的数量：“0”表示不允许创建淘专辑。
25）最多允许关注淘专辑的数量：允许用户最多关注的淘专辑数量。
2、论坛相关 => 帖子相关

1)允许发新话题：设置是否允许发新话题。注意：只有当用户组阅读权限高于 0 时，才能发新话题。
2)允许发表回复：设置是否允许发表回复。注意：只有当用户组阅读权限高于 0 时，才能发表回复。
3)允许直接发帖：是否允许直接发帖或者对回复及主题进行审核，此设置受版块审核影响，如果版块设置为不需要审核，则此设置无效。
4)允许下载远程图片：选择“是”远程图片将本地化保存。
5)允许发匿名帖：是否允许用户匿名发表主题和回复，只要用户组或本论坛允许，用户均可使用匿名发帖功能。匿名发帖不同于游客发帖，用户需要登录后才可使用，版主和管理员可以查看真实作者。
6)允许设置帖子权限：设置是否允许设置帖子需要指定阅读权限才可浏览。
7)主题(附件)最高售价：主题(附件)出售使得作者可以将自己发表的主题(附件)隐藏起来，只有当浏览者向作者支付相应的交易积分后才能查看主题(附件)内容。此处设置用户出售主题(附件)时允许设置的最高价格，0 为不允许用户出售。
注意：本功能需在 全局设置中启用交易积分后才可使用。如开启版主及管理员的出售主题(附件)功能，请将其最高价格设置为高于普通用户组允许的最高价格，这样在帖子被编辑时不至于导致主题(附件)售价的硬性下调。
8)允许使用 [hide] 代码：设置是否允许帖子中使用 [hide] 隐藏标签。
9）允许使用 [audio] [media] [flash] 等多媒体代码：允许使用 [audio] [media] [flash] 等多媒体代码，可以在帖子中发表mp3音乐，视频等多媒体文件，会员可直接观看。
10)允许签名中使用 Discuz! 代码：设置是否解析用户签名中的 Discuz! 代码。
11)允许签名中使用 [img] 代码：设置是否解析用户签名中的 [img] 代码。
12)签名文字最大长度：设置用户签名最大字节数，0 为不允许用户使用签名。
13)主题评价影响值：设置用户评价一次主题，对主题评价指数的影响，0 为不允许用户评价主题。
14)编辑帖子时间限制：帖子作者发帖后超过此时间限制将不能再编辑帖子，版主和管理员以及在【全局】=>【用户权限】=>【允许用户随时编辑的帖子类型】中被选中的帖子类型不受此限制，0 为不限制。
15）允许设置回帖奖励：允许用户在发布主题时给予回帖者一定的扩展积分奖励。回帖奖励默认积分可在【全局】=>【积分设置】=>【交易积分设置】中指定。
16）允许使用标签：设置是否允许发表新帖时使用标签。
17)帖子直接点评：设置用户允许点评的帖子。注意: 只有在【全局】=>【站点功能】的帖子点评设置中开启直接点评功能，本设置才会生效。
18）楼层回复点评：允许用户通过点击帖子中的“回复”链接回复他人回帖时，自动对该楼层产生点评
注意：只有在 全局 - 站点功能 - 帖子点评 中开启楼层回复功能，本设置才会生效；点击主题中的“回复”链接，不会对主题产生点评
19)允许用户在点评时发表点评观点：允许用户在点评时发表观点。注意：只有在【全局】=>【站点功能】的帖子点评设置中开启直接点评功能，本设置才会生效。
20）发帖时可@其他人的数量：0为不允许 @ 其他人（包括他关注的人和他的好友）
21）允许设置预发帖时间：允许用户在发布主题时设置指定的发帖时间。只能设置在每半小时（如 13:00 或 13:30）自动发布。
22）允许评论淘专辑：允许用户在主题和淘专辑页面进行评论。
3、论坛相关 => 附件相关

1)允许下载附件：设置是否允许在没有设置特殊权限的论坛中下载附件。
2）允许查看图片：设置是否允许在没有设置特殊权限的论坛中查看图片。
3)允许上传附件：设置是否允许上传附件到没有设置特殊权限的论坛中。需要 PHP 设置允许才有效，请参考管理中心首页
如下图所示：

4)允许设置附件权限：设置是否允许设置附件需要指定积分以上才可下载。
5)允许上传图片：设置是否允许上传图片到没有设置特殊权限的论坛中。需要 PHP 设置允许才有效，请参考管理中心首页同上。
6)论坛最大附件尺寸(单位K 1M=1024K)：设置用户上传的每个附件的最大尺寸。需要 PHP 设置允许才有效，请参考管理中心首页同上。
7)每天最大附件总尺寸(单位K 1M=1024K)：设置用户上传的每天总附件的总尺寸。注意：本功能会加重服务器负担，建议仅在必要时使用。
8)每天最大附件数量：设置用户每 24 小时可以上传的附件数量，0 为不限制。
9)允许附件类型：设置允许上传的附件扩展名，多个扩展名之间用半角逗号 "," 分割，留空为不限制。
4、论坛相关 => 特殊主题

1)允许发起活动：选择“是”允许在论坛发布活动主题。
2)允许发起投票：选择“是”允许在论坛发布投票主题。
3)允许参与投票：选择“是”允许在论坛参与投票。
4)允许发起悬赏：选择“是”允许在论坛发布悬赏主题，悬赏使用交易积分。
5)最小悬赏积分：每笔悬赏最少悬赏积分(最小设置为 1)。
6)最大悬赏积分：每笔悬赏最多悬赏的积分，最大可设置为 32767 (不限制请输入 0)。
7)允许发布商品：选择“是”允许在论坛发布商品主题。
8)最小交易金额：每笔交易的最小交易金额(单位：人民币 元，最小交易金额为 1 元)。
9)最大交易金额：设置每笔交易所能设置的最大交易金额(单位：人民币 元，不限制请输入 0)。
10)商品主题最多推荐商品数：设置用户在每篇商品主题中最多推荐商品的数目。
11)允许发起辩论：选择“是”允许在论坛发布辩论主题。
12)允许发表抢楼帖：选择“是”则抢楼帖不允许用户删除回帖。
5、群组相关

1)允许建立群组的数量：在这里可以设置允许改管理组建立多少个群组，范围1-255，0 为不允许创建。
2)允许直接发帖：设置是否允许直接发帖或者对回复及主题进行审核。
3)是否允许发站外URL：是否允许发帖包含URL，合理的设置可以有效的减少广告帖的量。

6、门户相关

1)文章评论字数：设置此管理组发表文章评论字数限制，设置为0将禁止此用户级发表评论。
2)发布文章：设置是否允许此管理组用户在门户发布文章。选择“是”可以选择允许下载本地图片和发布文章是否需要审核，如果允许下载本地图片，那么下载远程图片的同时允许重新生成一份本地图片。

7、空间相关

1)空间大小(单位M)：设置最大空间尺寸(单位M)，0 为不限制。
2)发表日志：是否允许在个人空间中发布日志，选择“是”则可以设置发表日志是否需要审核。
3)发表记录：是否允许在个人空间中发表记录，选择“是”则可以设置发表记录是否需要审核。
4)上传图片：是否允许上传照片，同时这个会影响论坛的编辑器上上传图片的功能。选择“是”则可以设置上传的图片是否需要审核。
5)发布分享：是否允许发布分享，选择“是”则可以设置发布分享是否需要审核。
6)允许打招呼：是否允许跟其他用户打招呼。
7)允许加好友：是否允许加其他用户为好友。
8)允许表态：是否允许对日志表态。
9)发表留言/评论：是否允许发表留言/评论。
10)允许使用漫游(Manyou)应用：是否允许使用漫游应用游戏。
11)不受视频认证限制：该管理组是否可以不受视频认证限制。
12)允许查看视频认证：该管理组用户是否允许查看视频认证信息。
13)允许自定义模块使用HTML：空间中添加的自定义模块是否允许使用 HTML 代码。
14)允许自定义模块使用BBCODE：设置是否解析个人主页自定义模块中的 BBCODE 代码。
15)允许自定义模块使用[img]：设置是否解析个人主页自定义模块中的 [img] 代码。

8、其他 => 积分相关

1)发短消息不扣积分：设置发短消息是否按照积分规则扣积分。
2)搜索不扣积分：设置搜索是否按照积分规则扣积分。
3)允许参与评分：允许该管理组用户在给其他用户评分的时候可以评哪种积分，以及可以评分的最大值与最小值，和24小时评分总量。
同时还可以设置扣除自身积分，及参与评分不扣系统的积分而是扣自身的积分。
如果不启用，该管理组没有评分功能。

9、其他 => 道具相关

道具相关设置需要打开道具功能方生效，操作路径：【后台】=>【运营】=>【道具中心】
1)允许使用道具：是否允许使用道具，包括购买、使用、转让的权限。
2)购买道具折扣：购买道具的折扣值，范围 1~10 数值越小折扣越多，0 为不享受折扣。
3)道具包容量：本组用户带有道具的最大重量，范围 0~60000。注：每个道具重量由道具设置决定。

10、其他 => 注册相关

邀请注册相关设置需要打开邀请注册功能方生效，操作路径：【后台】=>【全局】=>【注册与访问控制】开放邀请注册。
1)允许购买邀请码：是否允许购买邀请码。
2)允许发送邀请注册：是否允许通过站点邮件系统发送邀请码。
3)邀请码购买价格：该管理组的会员购买邀请码的价格。
4)24小时内购买邀请码最大数量：该管理组24小时内购买邀请码的最大数量，超过此数量将不能购买新的邀请码。
5)邀请码有效期：该管理组购买邀请码的最大有效期，超过此有效期验证码将自动失效，单位为天，默认为10。

二、管理权限


点击管理组后面的“管理权限”，可以进入管理权限设置页面。
管理权限包括主题管理权限、帖子管理权限、管理面板权限、空间管理权限、门户管理权限和其它权限。

1、主题管理权限
1)允许置顶主题：设置是否允许置顶管理范围内主题的级别。如果打开全局置顶的功能，置顶 III 将全论坛置顶，置顶 II 将在当前分类置顶。
2)允许精华主题：设置是否允许精华管理范围内主题的级别。
3)允许提升主题：设置是否允许提升管理范围内的主题。
4)允许高亮主题：设置是否允许高亮管理范围内的主题。
5)允许推荐主题：设置是否允许推荐管理范围内的主题 (要在相应版块的扩展设置里开启推荐主题功能) 。
6)允许添加主题图章：设置是否允许给管理范围内的主题在帖子内容页添加图章。
7)允许添加主题图标：设置是否允许给管理范围内的主题在帖子列表页添加图标。
8)允许关闭主题：设置是否允许关闭管理范围内的主题。
9)允许移动主题：设置是否允许移动管理范围内的主题。
10)允许编辑主题分类：设置是否允许编辑管理范围内主题的分类。
11)允许复制主题：设置是否允许复制管理范围内的主题。
12)允许合并主题：设置是否允许合并管理范围内的主题。
13)允许分割主题：设置是否允许分割管理范围内的主题。
14)允许修复主题：设置是否允许修复管理范围内的主题。
15)允许强制退款：允许将作者发布的收费主题终止，并将作者所获得收益退还给购买者。退款后，该主题将不能再被设置为收费主题，同时作者将自行承担积分交易而产生的损失。
16)允许编辑投票：设置是否允许编辑管理范围内投票主题的选项。
17)允许移除悬赏：设置是否允许移除管理范围内悬赏主题的悬赏金额。
18)允许管理活动报名者：设置是否允许管理范围内活动主题的报名者。
19)允许编辑商品：设置是否允许编辑管理范围内商品主题的商品。
20）允许添加用户标签：设置是否允许编辑管理范围内主题的用户标签。

2、帖子管理权限

1)允许编辑帖子：设置是否允许编辑管理范围内的帖子。
2)允许警告帖子：设置是否允许警告管理范围内的帖子。
3)允许屏蔽帖子：设置是否允许屏蔽管理范围内的帖子。
4)允许删除帖子：设置是否允许删除管理范围内的帖子。
5)允许置顶回帖：设置是否允许置顶管理范围内的回帖。
6）允许管理标签：设置是否允许管理管理范围内的标签。

3、管理面板权限

1)允许审核帖子：设置是否允许审核用户发表的帖子，只在站点设置需要审核时有效。
2)允许审核用户：设置是否允许审核新注册用户，只在站点设置需要人工审核新用户时有效。
3)允许禁止用户发言：设置是否允许禁止用户发帖。
4）允许禁止用户访问：设置是否允许禁止用户访问站点。
5)允许禁止 IP：设置是否允许添加或修改禁止 IP 设置。
6)允许编辑用户：设置是否允许编辑用户资料。
7)允许批量删帖：设置是否允许通过管理面板批量删帖，版主在面板中批量删帖的范围是 1 周内，超级版主删帖范围是 2 周内，管理员无限制。
8)允许编辑版块：设置是否允许编辑管理范围内的版块的资料。
9)允许发布公告：设置是否允许发布站点公告。
10)允许删除回收站的帖子：设置是否允在“前台->管理面版->主题回收站”中删除主题。
11)允许查看管理日志：设置是否允许查看管理日志。

4、空间管理权限

1)允许管理动态(feed)：是否允许管理空间中产生的 feed。
2)允许管理记录：是否允许管理用户发表的记录。
3)允许管理分享：是否允许管理用户的分享。
4)允许管理日志：是否允许管理用户的日志。
5)允许管理相册：是否允许管理用户的相册。
6)允许管理评论：是否允许管理用户的评论。
7)允许管理道具记录：是否允许管理道具记录。
8)允许管理举报：是否允许管理举报信息。
9)允许管理推荐成员：是否允许管理推荐成员。
10)允许管理推荐好友：是否允许管理推荐好友。
11)允许管理视频认证：是否允许管理视频认证。
12)允许管理道具：是否允许管理道具。
13)允许管理表态动作：是否允许管理表态动作。

5、门户管理权限


1)允许管理所有文章：是否允许管理门户文章。选择“是”则允许此管理组成员管理编辑删除所有频道栏目文章，
2)允许创建专题：是否允许创建门户专题。
3)允许管理专题：设置是否允许编辑所有专题。
4)允许 DIY：设置是否允许 DIY 论坛，文章，空间，论坛等页面。

6、其他权限

这里可以设置是否允许查看用户 IP，以及是否允许管理淘专辑。
