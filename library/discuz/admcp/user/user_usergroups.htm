用户组

用户组分为会员用户组、自定义用户组和系统用户组，会员组以积分确定组别和权限，而系统组和自定义用户组是人为设定，不会由系统自行改变。系统组和自定义用户组的设定不需要指定积分，Discuz! 预留了从站点管理员到游客等的 8 个系统头衔，自定义组的用户需要在编辑会员时将其加入。会员用户组至少分为两组，其积分下限分别为负值(任一负数)和 0，否则将导致部分用户无法与用户组匹配的问题。
操作路径：【后台】=>【用户】=>【用户组】

一、3种用户组介绍


1、会员用户组
会员用户组是普通会员的用户组，该用户组以积分确定组别和权限。在此处可以设置各个用户组需要的最低积分，提交修改的时候系统会自动修改上一用户组的最高积分。点击用户组后面的详情可以设置该用户组的详细权限。

2、自定义用户组
自定义用户组是管理员手工创建的用户组。
点击“添加新自定义用户组”，可以添加一个新的用户组，可以自定义组头衔，星星数，头衔颜色。
自定义用户组的用户需要在编辑会员时将其加入，用户被加入自定义用户组后，将不受会员用户组级别的限制。可以点击“查看”来查看当前有多少用户在这个自定义用户组里面，点击“编辑”来设置自定义用户组的权限。

3、系统用户组
系统预留了从站点管理员到游客等的 8 个系统头衔，点击“系统用户组”，可以进入系统用户组设置页面。

在这里，可以修改几个默认的系统用户组的名称与星星数目等，同样，点击“编辑”可以看到用户组的具体设置。

二、用户组详细权限设置


现在以我们在自定义用户组中添加的“VIP用户”用户组为例，讲解一下用户组的详细设置。
操作路径：【后台】=>【用户】=>【用户组】=>【自定义用户组】，添加了一个名为“VIP用户”的自定义用户组之后，点击“编辑”进行详细权限设置
1、基本设置

1)用户组头衔：设置用户组头衔名称。
2)组图标：设置该用户组的图标，可以选择是输入 URL 或者是上传图片，使用方法同管理组中的组图标一样。
3)访问站点权限
禁止访问：禁止用户浏览本站任何页面。
正常访问：当站点开放的时候，可以正常浏览页面。
超级访问：即便站点关闭也可以访问，一般用于管理员或站内测试组。
4)阅读权限：设置用户浏览帖子或附件的权限级别，范围 0～255，0 为禁止用户浏览任何帖子或附件。当用户的阅读权限小于帖子或附件的阅读权限许可(默认时为 1)时，用户将不能阅读该帖子或下载该附件。
5)最多好有数：该用户组的用户能够拥有的最多好友数量，0为无限。
除了用户组的好友数限制外，如果想单独给某个用户组的成员单独增加好友数，可以到【用户】=>【用户管理】=>【详情】中添加额外好友数。
6)二级域名最短长度：站点开启二级域名功能后有效，0为禁止使用二级域名。
7)允许隐身：是否允许用户在站点上隐身。
8)允许积分转账：设置是否允许用户在个人中心将自己的交易积分转让给其他用户。注意：本功能需在 全局设置中启用交易积分后才可使用。
9)允许发送短消息：设置是否允许用户发送短消息。选择“是”后可以设置24小时内发布短消息的最大数及是否可以给任何人发短消息，选择不允许给任何人发短消息，那么选择“否”当对方设置为只接受好友短消息，将无法对其发送短消息。
10)允许使用 HTML 代码：是否允许用户发布信息的时候使用 HTML 代码。注意：开放 HTML 功能将产生安全隐患，请慎用。建议只在十分必要的情况下使用，并限制只开放给最核心的管理人员
11）允许发站外URL：该项设置可以有效减少广告帖子。
12)允许查看站点统计：是否允许用户查看站点统计。选择“是”用户在访问站点的时候页面底部会出现“站点统计”链接，点击即可查看。

13)允许全文搜索：是否允许用户进行全文搜索，此设置只对搜索论坛有效。注意：当数据量大时，全文搜索将非常耗费服务器资源，请慎用。
14)允许使用搜索：选择为该用户组用户开启哪些模块的搜索。包括搜索文章、论坛、日志、相册和群组。
15)操作理由短消息通知作者：设置用户在对他人评分或管理操作时是否强制输入理由和通知作者。
16)允许自定义头衔：设置是否允许用户设置自己的头衔名字并在帖子页面中显示。选择“是”用户在【个人资料】中设置自己的头衔后效果如下：

17)不受时间段限制：设置用户是否可以不受全局设置中设置的特殊时间段禁令限制。
18)会员每小时发主题数限制：设置允许会员每小时最多的发表的主题数量，可以配合灌水预防功能进一步限制会员的发帖，可设置为 1～255 范围内的数值，0 为不限制。此功能会轻微加重服务器负担, 且对游客无效。
19）会员每小时发回帖数限制：设置允许会员每小时最多的发回帖数量，可以配合灌水预防功能进一步限制会员的发帖，可设置为 1～255 范围内的数值，0 为不限制。此功能会轻微加重服务器负担，且对游客无效。
20)启用防灌水验证码或验证问答机制：开启防灌水验证机制可以防止灌水机等，但会增加用户操作易用度。修改密码不受此限制，开启防灌水验证码或验证问答后，该设置有效。
21)发表不受限制：设置发表是否不受灌水预防和最大字数限制等。
22)忽略需要审核的关键字：选择“是”则此用户组的用户在发表时不会触发需要审核的关键字。
23）允许用户创建淘专辑的数量：“0”表示不允许创建淘专辑。 24）最多允许关注淘专辑的数量：允许用户最多关注的淘专辑数量。

2、用户组体制
特殊用户组体制分为“非公众用户组”、“公众用户组”、“收费公众用户组”、“免费公众用户组”。根据不同的用户组体制设置来决定的。

1)公众用户组：设置是否为公众用户组。注意：使用此功能前请确定是否确实需要，否则用户一旦自行加入本组，即便之后再设置本组为非公众用户组，仍然无法将用户从该组中批量去除，除非删除本用户组。公众用户组不能关联管理组，以免产生安全隐患。
2)非公众用户组：
选择“否”为非公众用户组，当用户组设置为“非公众用户组”时，无论是以主用户组的形式，还是以扩展用户组的形式，均只能由管理员手工将用户加入本组。
选择“是”会出现“收费公众用户组日价格”和“收费公众用户组一次最短购买时间”的设置
3)收费公众用户组日价格：如果您需要用户支付相应的交易积分后才能加入本用户组，请在此输入本组的日购买价格，单位为交易积分所使用的单位。例如设置为 2，则用户购买为期 30 天的本组权限，需要支付 2*30=60 个交易积分，30 天有效期后该用户将自动从本组中退出。如设置为 0，则本组为免费公众用户组。注意: 本功能需要首先设定用户组为公众用户组且设置了有效的交易积分后才有效。
4)收费公众用户组一次最短购买时间(天)：设置用户在购买本组权限时，一次支付价值至少多少日的交易积分，请设置为大于等于 1 的数值。例如日价格为 2，最短购买时间为 10，则用户至少需要花费 2*10=20 个交易积分，获得至少 10 天本组权限，才能加入本组。注意: 本功能需要首先设定用户组为收费公众用户组才有效。

3、论坛相关 => 帖子相关
1)允许发新话题：设置是否允许发新话题。注意：只有当用户组阅读权限高于 0 时，才能发新话题。
2)允许发表回复：设置是否允许发表回复。注意：只有当用户组阅读权限高于 0 时，才能发表回复。
3)允许直接发帖：是否允许直接发帖或者对回复及主题进行审核，此设置受版块审核影响，如果版块设置为不需要审核，则此设置无效。
4)允许下载远程图片：选择“是”远程图片将本地化保存。
5)允许发匿名帖：是否允许用户匿名发表主题和回复，只要用户组或本论坛允许，用户均可使用匿名发帖功能。匿名发帖不同于游客发帖，用户需要登录后才可使用，版主和管理员可以查看真实作者。
6)允许发匿名帖：是否允许用户匿名发表主题和回复，只要用户组或本论坛允许，用户均可使用匿名发帖功能。匿名发帖不同于游客发帖，用户需要登录后才可使用，版主和管理员可以查看真实作者。
7)允许设置帖子权限：设置是否允许设置帖子需要指定阅读权限才可浏览。
8)主题(附件)最高售价：主题(附件)出售使得作者可以将自己发表的主题(附件)隐藏起来，只有当浏览者向作者支付相应的交易积分后才能查看主题(附件)内容。此处设置用户出售主题(附件)时允许设置的最高价格，0 为不允许用户出售。
注意：本功能需在 全局设置中启用交易积分后才可使用。如开启版主及管理员的出售主题(附件)功能，请将其最高价格设置为高于普通用户组允许的最高价格，这样在帖子被编辑时不至于导致主题(附件)售价的硬性下调。
9)允许使用 [hide] 代码：设置是否允许帖子中使用 [hide] 隐藏标签。
10）允许使用 [audio] [media] [flash] 等多媒体代码：允许使用 [audio] [media] [flash] 等多媒体代码，可以在帖子中发表mp3音乐，视频等多媒体文件，会员可直接观看。
11)允许签名中使用 Discuz! 代码：设置是否解析用户签名中的 Discuz! 代码。
12)允许签名中使用 [img] 代码：设置是否解析用户签名中的 [img] 代码。
13)签名文字最大长度：设置用户签名最大字节数，0 为不允许用户使用签名。
14)主题评价影响值：设置用户评价一次主题，对主题评价指数的影响，0 为不允许用户评价主题。
15)编辑帖子时间限制：帖子作者发帖后超过此时间限制将不能再编辑帖子，版主和管理员以及在【全局】=>【用户权限】=>【允许用户随时编辑的帖子类型】中被选中的帖子类型不受此限制，0 为不限制。
16）允许设置回帖奖励：允许用户在发布主题时给予回帖者一定的扩展积分奖励。回帖奖励默认积分可在【全局】=>【积分设置】=>【交易积分设置】中指定。
17）允许使用标签：设置是否允许发表新帖时使用标签。
18)帖子直接点评：设置用户允许点评的帖子。注意: 只有在【全局】=>【站点功能】的帖子点评设置中开启直接点评功能，本设置才会生效。
19）楼层回复点评：允许用户通过点击帖子中的“回复”链接回复他人回帖时，自动对该楼层产生点评
注意：只有在 全局 => 站点功能 => 帖子点评 中开启楼层回复功能，本设置才会生效；点击主题中的“回复”链接，不会对
主题产生点评
20)允许用户在点评时发表点评观点：允许用户在点评时发表观点。注意：只有在【全局】=>【站点功能】的帖子点评设置中开启直接点评功能，本设置才会生效。
21）发帖时可@其他人的数量：0为不允许 @ 其他人（包括他关注的人和他的好友）
22）允许设置预发帖时间：允许用户在发布主题时设置指定的发帖时间。只能设置在每半小时（如 13:00 或 13:30）自动发布。
23）允许评论淘专辑：允许用户在主题和淘专辑页面进行评论。

2)允许发起投票：选择“是”允许在论坛发布投票主题。

6)最大悬赏积分：每笔悬赏最多悬赏的积分，最大可设置为 32767 (不限制请输入 0)。
7)允许发布商品：选择“是”允许在论坛发布商品主题。
8)最小交易金额：每笔交易的最小交易金额(单位：人民币 元，最小交易金额为 1 元)。

10)商品主题最多推荐商品数：设置用户在每篇商品主题中最多推荐商品的数目。
11)允许发起辩论：选择“是”允许在论坛发布辩论主题。
12)允许发表抢楼帖：选择“是”则抢楼帖不允许用户删除回帖。

6、群组相关

1)允许建立群组的数量：在这里可以设置允许该用户组建立多少个群组，范围1-255，0 为不允许创建。
2)允许直接发帖：设置是否允许直接发帖或者对回复及主题进行审核。
3)是否允许发站外URL：是否允许发帖包含URL，合理的设置可以有效的减少广告帖的量。

7、门户相关

7)允许加好友：是否允许加其他用户为好友。

5)可直接查看收费主题：设置是否可以直接查看收费的主题。
6)允许参与评分：允许该用户组用户在给其他用户评分的时候可以评哪种积分，以及可以评分的最大值与最小值，和24小时评分总量。
同时还可以设置扣除自身积分，及参与评分不扣系统的积分而是扣自身的积分。
如果不启用，该用户组没有评分功能。

10、其他 => 道具相关

道具相关设置需要打开道具功能方生效，操作路径：【后台】=>【运营】=>【道具中心】
1)允许使用道具：是否允许使用道具，包括购买、使用、转让的权限。
2)购买道具折扣：购买道具的折扣值，范围 1~10 数值越小折扣越多，0 为不享受折扣。
3)道具包容量：本组用户带有道具的最大重量，范围 0~60000。注：每个道具重量由道具设置决定。

11、其他 => 注册相关

邀请注册相关设置需要打开邀请注册功能方生效，操作路径：【后台】=>【全局】=>【注册与访问控制】开放邀请注册。
1)允许购买邀请码：是否允许购买邀请码。
2)允许发送邀请注册：是否允许通过站点邮件系统发送邀请码。
3)邀请码购买价格：该用户组的会员购买邀请码的价格。
4)24小时内购买邀请码最大数量：该用户组24小时内购买邀请码的最大数量，超过此数量将不能购买新的邀请码。
5)邀请码有效期：该用户组购买邀请码的最大有效期，超过此有效期验证码将自动失效，单位为天，默认为10。

三、用户组批量编辑


批量编辑用户组，即在要编辑用户组的时候，选择多个用户组，然后点击批量编辑，就可以进入批量编辑界面。
按住 alt 键修改那些单选，复选的项目，可以横向的全部选择。
从这里，就可以方便的批量设置多个用户组了。

点击"批量编辑"后就可以批量设置用户组的项目了。

四、复制设置


设置复制可将源用户组栏目的某些设置应用到其他多个用户组，用于以一个用户组的设置为基础，快速设置一批用户组。

目标用户组：指将源用户组复制到哪些目标用户组，可以按住 CTRL 多选。
项目选择：指要将源用户组的哪些设置复制到目标用户组，可按住 CTRL 多选。
用户组设置复制一旦提交立即生效，并无法恢复，请仔细选择目标用户组和设置项目。
