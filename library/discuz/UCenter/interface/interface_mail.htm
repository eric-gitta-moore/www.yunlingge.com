邮件接口

此部分接口用于添加邮件到队列。

添加邮件到队列

mixed uc_mail_queue(stringuids, stringemails, stringsubject, stringmessage[, mailfrommail, stringcharset, booleanhtmlon, integerlevel]);

函数参数
参数 含义
string uids 用户 ID 多个用逗号(,)隔开
string emails 目标email，多个用逗号(,)隔开
string subject 邮件标题
string message 邮件内容
mail frommail 发信人，可选参数，默认为空，uc后台设置的邮件来源作为发信人地址
string charset 邮件字符集，可选参数，默认为gbk
boolean htmlon 是否是html格式的邮件，可选参数，默认为FALSE，即文本邮件
integer level 邮件级别，可选参数，默认为1，数字大的优先发送，取值为0的时候立即发送，邮件不入队列
返回值
值 含义
mixed false:失败：进入队列失败，或者发送失败
integer:成功：进入队列的邮件的id，当level为0，则返回1

本函数用来将邮件加入到邮件队列，或者直接发送邮件（当level为0时）
邮件队列使用示例 (PHP)
$res = uc_mail_queue($uid, '', $_POST['subject'], $_POST['message']);
if(empty($res)) {
	echo '加入邮件队列失败';
} else {
	echo '加入邮件队列成功';
}
