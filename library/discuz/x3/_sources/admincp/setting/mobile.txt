手机版访问设置
==============================

**手机版设置** 

 




现在使用手机上网的用户越来越多，如果站长希望支持用户使用手机访问论坛，可以在【后台】=>【全局】=>【手机版访问设置】中开启论坛手机版。
在手机版访问的具体设置中，可以设定用户通过手机浏览器访问站点时，在用户手机上的显示方式，如：手机浏览器访问时自动跳转手机版、是否允许手机版进行注册、是否在手机版中开启验证码、在手机上的显示版式、每页主题数等。

当您在后台开启手机版后，用户使用手机访问论坛时，将自动跳转到手机版界面。

如果您在开启手机版后，想用独立的域名，可以进入：全局 => 域名设置 => 应用域名中进入配置

同时也可以在界面 => 导航设置=> 底部导航中开启手机版的底部导航链接入口，默认是开启的。

1、是否允许手机版注册

开启后，用户可以直接在手机上注册论坛会员帐号。

2、是否开启验证码

开启后，用手机访问论坛的用户进行注册、登录、发布等需要验证码，验证码开关：全局 => 防灌水设置 => 验证码设置

3、是否开启热帖

开启后则在首页显示热贴。

4、是否显示置顶帖

开启后在帖子列表页面中显示置顶贴。

5、开启不显示图片

默认显示图片，开启后，“触屏版”不显示帖子图片，“标准版”不显示帖子图片、小图标、主题列表日期等页面元素。

6、每页显示主题数

设置主题列表页每页显示主题的个数。

7、主题内每页显示帖数

设置主题内每页显示的帖子数目，推荐值为5。

8、页面浏览缓存时间

设置访问过的页面在用户手机上保存的时间，留空或0为不缓存

9、首页分区展示方式

选择“收起”，论坛首页将默认只展示分区标题用户需点击分区展开该分区下的版块列表。

选择“展开”，论坛首页将默认展示分区即分区下的所有版块列表。

10、手机发帖来源自定义

如果填写，将在看帖页面“来自手机”处增加自定义文字或图标。

如：<a href='http://m.x.com'>手机频道</a>此处支持HTML标签有：<a><font><span><strong><b><img>，如果使用图标，请保证图片高度在16px(像素)以内。

11、是否启用极简版

极简版为一些仅支持WML语言的较早期的设备浏览网站的内容。
